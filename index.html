Three of 14 posters for Macintosh’s 20th birthday present its groundbreaking GUI:


Coming after Jaguar (Mac OS X 10.2), Panther was supposed to be designated Mac OS X Version 11, but Apple eventually settled on 10.3. A CD picture with version 11 label was available for a while on Apple’s public relation subsite.
This most important update to Macintosh’s operating system since 1984 included wide use of colour, new icons, scalable fonts, balloon help, Apple Menu customization, labels and aliases.




Just one look at desktops from Windows 1.0, Windows 95 and Windows XP simultaneously can tell more about the evolution of GUIs and graphic hardware than hours of studying specifications. You can also check out desktops from other operating systems.




The icons for clock usually show some fixed hours, such as 3 PM in Windows 3.x and five past five in OS/2 Warp. This icon, from Windows 1.0, is an exception and always shows the current system time.
